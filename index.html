Overview

SLAM-Dunk-Prometheus is a comprehensive robotics perception system that combines real-time LiDAR-IMU SLAM with state-of-the-art 3D semantic segmentation. Our pipeline enables accurate localization, mapping, and scene understanding for autonomous systems and robotics applications.

The system consists of two main components: a robust SLAM pipeline for real-time pose estimation and map building, and an advanced segmentation module that provides semantic understanding of 3D environments using cutting-edge deep learning models.

System Modules

SLAM Pipeline

Real-time LiDAR-IMU SLAM system optimized for the StrayScanner dataset. Features include:

  • High-frequency LiDAR and IMU sensor fusion
  • Real-time pose estimation and trajectory optimization
  • Efficient point cloud processing and map building
  • Comprehensive data preprocessing and post-processing tools
Python LiDAR IMU Real-time

3D Segmentation

Advanced 3D semantic segmentation using state-of-the-art deep learning models. Capabilities include:

  • Mosaic3D and OpenScene model integration
  • Real-time point cloud semantic labeling
  • Interactive visualization tools for segmented scenes
  • Comprehensive demo codes for easy deployment
Python Deep Learning 3D Vision Semantic Segmentation

Technical Highlights

Performance

Optimized for real-time operation with efficient algorithms and parallel processing. Achieves high frame rates while maintaining accuracy in pose estimation and segmentation.

Dataset Support

Built specifically for the StrayScanner dataset but extensible to other LiDAR-IMU datasets. Includes comprehensive preprocessing tools for data preparation.

Modular Design

Separated SLAM and segmentation modules allow flexible integration. Each component can be used independently or combined for complete scene understanding.

Open Source

All code is publicly available on GitHub under permissive licenses. Comprehensive documentation and demo codes facilitate easy adoption.
